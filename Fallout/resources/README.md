# Ressources — Fallout Campaign

> Ce dossier contient les ressources externes de ta campagne :
> PDFs de scénarios, cartes, documents officiels, notes de prep.

---

## Comment ajouter une ressource

1. Dépose le fichier ici (`resources/nom_du_fichier.pdf`)
2. Ajoute une entrée dans l'index ci-dessous
3. Indique à Claude de lire le fichier pour l'intégrer au contexte

## Types de ressources recommandés

### Scénarios
- PDFs d'aventures officielles Modiphius
- Scénarios homebrew récupérés en ligne
- Tes propres preps de session

### Lore & Référence
- Extraits du Core Rulebook (règles spéciales)
- Lore régional spécifique à ta campagne
- Cartes de zones

### Outils
- Tables de rencontres custom
- Listes d'équipement spécifiques
- Bestaires supplémentaires
